# KERSOG Compass

Mobile-first prototype for Compass, a field app for roofing salespeople.

## Current screens
- Welcome
- Create Account
- Sign In
- Dashboard
- New Claim
- Upload EagleView / documents
- Roof Inspection
- Photos
- Estimate
- Reports

## How to publish
Use GitHub Pages:
Settings → Pages → Deploy from branch → main → /(root)

## Brand assets
Replace `assets/images/kersog-emblem.svg` with the real KERSOG emblem when ready.
