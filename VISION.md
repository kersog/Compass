# Why Compass Exists

Roofing professionals spend too much time fighting software.

Compass exists to remove that friction.

The goal is not simply to generate estimates.

The goal is to guide professionals through the entire inspection process in a way that naturally produces better documentation, stronger claims, and more consistent results.

Compass should feel like an experienced mentor standing beside the user.

It should guide—not overwhelm.

Every screen should reduce mental effort.

Every tap should move the user closer to completing the inspection.

Compass should become the easiest roofing field software available.

Not because it has fewer capabilities.

Because those capabilities are organized in the exact order that roofing professionals actually work.

The software should be beautiful.

Professional.

Trustworthy.

Fast.

Confident.

Calm.

As Compass grows, every new feature should strengthen this mission rather than complicate it.

When making decisions, ask:

Does this make the field user's job easier?

Does this reduce confusion?

Does this improve documentation?

Does this improve claim quality?

If not...

It probably does not belong in Compass.

Compass is more than software.

It is a field operating system for roofing professionals.

Built by people who understand roofing.

Built for people who climb roofs every day.

Built with purpose.
