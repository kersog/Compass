**Product Owner:** Bob Goodrum

**Company:** KERSOG

**Product:** Compass

**Current Version:** MVP 1.0

---

# Product Vision

Compass is a premium field application built specifically for roofing professionals.

Its purpose is to guide a user from the first property inspection through complete claim documentation, estimate generation, and report export.

Compass should remove complexity while increasing consistency.

Every screen should help a field user perform the next task.

---

# Primary Users

• Roofing Sales Representatives

• Project Managers

• Roofing Contractors

• Independent Adjusters

• Insurance Field Adjusters

Future:

• Appraisers

• Desk Review Teams (Framework)

---

# Design Philosophy

Compass should feel calm.

Compass should feel premium.

Compass should never feel cluttered.

Every screen has one purpose.

Large touch targets.

Minimal text.

Fast workflows.

Everything should be optimized for phones first and tablets second.

Desktop support is optional.

# Guiding Principle

Every feature should answer one question:

"Does this help someone standing at the property?"

If the answer is no...

It probably belongs in Framework instead of Compass.
